# Original Template Integration

이 묶음은 사용자가 업로드한 기존 Markdown 템플릿을 PulpitInk 프로젝트에 맞게 통합한 결과입니다.

## 반영한 방향

- 기존 `AGENTS.md`의 Automation First 원칙을 유지했습니다.
- 기존 `AGENTS.md`의 Safety Rails를 PulpitInk의 라이선스, 로컬 데이터, YouTube 후순위 정책에 맞게 조정했습니다.
- 기존 `HISTORY.md`와 `CHANGELOG.md` 구조를 유지했습니다.
- 기존 릴리즈 산출물 검증 규칙을 Windows EXE/ZIP 중심으로 변환했습니다.
- 기존 모바일 배포 관련 규칙은 PulpitInk v1.0 범위에서는 제외하고, 필요 시 후순위 문서에서 다루도록 조정했습니다.

## 보존한 원본

업로드 원본은 `original_templates/` 폴더에 보존했습니다.
