SermonScript

설교·강의·회의 녹음을 로컬에서 텍스트로

faster-whisper와 FFmpeg 기반의 Windows 데스크톱 STT 앱입니다. 긴 녹음을 전처리하고, 세그먼트별로 검수한 뒤 TXT, JSON, Markdown, SRT, VTT, CSV로 내보냅니다.

Portable ZIP 받기 README 보기 사용자 가이드
100% 로컬 처리 오디오와 텍스트를 기본적으로 외부 서버에 보내지 않습니다.
Windows 데스크톱 GUI와 CLI를 함께 제공하고 같은 core 로직을 재사용합니다.
설교 특화 후처리 성경 구절 정규화, 사용자 사전, 원문 대조 후보를 지원합니다.
다중 포맷 출력 TXT, JSON, Markdown, SRT, VTT, CSV 결과물을 생성합니다.

검수까지 생각한 로컬 STT 워크플로우

SermonScript는 단순히 음성을 텍스트로 바꾸는 데서 멈추지 않습니다. 원본을 보존하고, 사람이 확인해야 하는 부분을 남기며, 편집 결과가 export에 반영되도록 설계했습니다.

긴 녹음 처리

30분 이상 설교 녹음을 전처리하고 세그먼트 단위 STT 결과를 생성합니다.

원본 보존

오디오 원본과 STT 원문 raw_text를 수정하지 않고 edited_text를 별도 저장합니다.

한국어 교정 후보

성경 용어, 사용자 사전, Jamo fuzzy 매칭을 활용해 pending 후보를 제안합니다.

편집기

타임스탬프, 검색·치환, 확인 필요 표시, 교정 후보 적용/무시를 제공합니다.

SQLite 기록

작업, 세그먼트, export, 원문 대조, 교정 후보를 로컬 DB에 보존합니다.

릴리즈 자동화

태그 push로 Windows Portable ZIP과 SHA256SUMS를 GitHub Release에 첨부합니다.

PowerShell에서 바로 시작

개발 환경에서는 소스 설치 후 GUI 또는 CLI로 실행할 수 있습니다. FFmpeg는 PATH에서 실행 가능해야 하며, STT 모델은 최초 사용 시 사용자 데이터 폴더에 캐시됩니다. 

git clone https://github.com/jeiel85/sermon-script.git
cd sermon-script
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -e ".[gui,reference,dev]"

sermonscript doctor
python -m sermonscript.app.main
  • 지원 입력: mp3, wav, m4a, aac, flac, ogg, mp4 등 로컬 파일
  • 지원 출력: txt, json, md, srt, vtt, csv
  • v1.0 제외: YouTube URL, 온라인 다운로드, 클라우드 동기화, 실시간 녹음
  • 주요 플랫폼: Windows 10/11, Python 3.11+
