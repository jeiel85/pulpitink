# Product Spec: SermonScript

## 1. 제품 정의

SermonScript는 설교, 강의, 회의 녹음 파일을 로컬 PC에서 전처리하고 텍스트로 변환한 뒤, 검수·편집·출력까지 지원하는 오픈소스 데스크톱 STT 프로그램입니다.

## 2. 핵심 사용자

| 사용자 | 사용 목적 |
|---|---|
| 목회자 | 설교 녹음 원고화 |
| 교회 미디어팀 | 설교 자막, 블로그, 주보 자료 제작 |
| 강사 | 강의 녹음 텍스트화 |
| 연구자 | 인터뷰 녹취 |
| 일반 사용자 | 회의/메모 녹음 변환 |

## 3. 제품 목표

- 30~90분 길이의 긴 오디오를 안정적으로 처리합니다.
- STT 전 오디오 전처리로 인식 품질을 보완합니다.
- STT 결과를 사용자가 빠르게 검수할 수 있게 합니다.
- 설교 원문, 성경본문, 사용자 사전을 활용해 보정 후보를 제안합니다.
- 결과를 TXT, Markdown, SRT, VTT, JSON, CSV로 출력합니다.

## 4. v1.0 기능 범위

### 포함

- 로컬 오디오/비디오 파일 입력
- 오디오 분석
- 오디오 전처리 프리셋
- faster-whisper STT 변환
- 작업 이력 저장
- 타임스탬프 편집
- 사용자 사전
- 성경 구절 보정
- 설교 원문 대조 변환
- Export
- Windows 배포

### 제외

- YouTube URL 입력
- 온라인 다운로드
- 클라우드 동기화
- 실시간 녹음
- 모바일 앱
- 자동 요약
- 화자 분리
- 자동 업데이트 설치

## 5. 제품 원칙

```text
원본 보존
로컬 우선
검수 중심
엔진 교체 가능
라이선스 투명성
긴 파일 안정성
```

## 6. 성공 기준

- 사용자가 30분 이상의 설교 녹음 파일을 불러와 변환할 수 있습니다.
- 전처리 후 STT가 실패하지 않고 세그먼트 단위 결과를 생성합니다.
- 사용자가 타임스탬프별로 텍스트를 수정할 수 있습니다.
- 수정본이 Export에 반영됩니다.
- 원본 파일은 수정되지 않습니다.
