# SermonScript 3-Goal Integrated Design Bundle

이 묶음은 SermonScript 통합 설계서에 3단계 대형 바이브 코딩 `/goal` 프롬프트를 반영한 버전입니다.

주요 변경:

1. `docs/vibe-coding-goals.md`를 3개 대형 Goal로 재구성
2. `docs/vibe-coding-goals-3phase.md` 추가
3. `README.md`, `HISTORY.md`, `CHANGELOG.md` 갱신
4. 기존 통합 설계서, AGENTS 규칙, YouTube deferred 문서, 초기 코드 스캐폴드 유지

권장 실행 순서:

```text
Goal 1 → Goal 2 → Goal 3
```

각 Goal은 에이전트가 2~3시간 이상 작업할 수 있는 큰 단위로 설계했습니다.
